# Open Source Thrust Analyzer

This project is an open-source alternative to TG20 thrust analyzer.

## Features
- Thrust measurement
- Voltage and current monitoring
- RPM measurement
- Live plots
- CSV data logging

## Hardware
- Arduino
- Load cell + HX711
- ESC + BLDC motor

## Software
- Python
- PyQt5
- Matplotlib
